seismic Web

Seismic Web is a web application that utilizes machine learning to detect earthquakes from audio files.

Key Features:
• Upload audio files (MP3, WAV, MSEED) for analysis.
• Visualize audio data (waveform) with Chart.js.
• Download detected earthquake events as MSEED files.
• Save waveform charts as PNG images.

Technical Details:
• Built with Flask (Python web framework).
• Leverages machine learning for earthquake detection (model training required).
• Utilizes libraries like Librosa, Obspy, Scikit-learn for audio processing and analysis.
• Employs Axios (frontend) and Chart.js (frontend) for user interaction and data visualization.
